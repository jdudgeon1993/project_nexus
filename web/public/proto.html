Proto v3 — swipeable card + FAB overlays

One header element instead of three. Schedule timeline is the only content in the sheet — everything else is a full-screen overlay launched from the FAB.

The route card (replaces header + direction toggle + tabs)

  • One card per direction. Swipe left/right (or click the › arrow) to switch direction — sky for outbound, purple for return, matching the live-vehicle colors.
  • Dots below show which direction is active.
  • ✕ closes the route entirely, back to Phase 1 map.

Timeline = the whole sheet

  • Tap the 🔍 icon above the timeline to reveal an inline filter — type "Peoria" to highlight/filter stops without leaving the screen.
  • Tap any stop card to expand live detail (vehicle, delay, accessibility) — same as before.

FAB → full-screen overlays

  • Alerts/Directions/Plan/Settings now slide up as full screens with their own back arrow — they get the entire screen when used, then return you exactly where you left off.
  • Alerts overlay keeps search + category pills + clickable route badges.
  • Tapping ⚠️3 on the map jumps straight to the Alerts overlay.

Map (Phase 3)

  • Selecting a direction in the card re-colors the map's route line/markers (sky ↔ purple) to match — try swiping the card.

Try it

  • Search "A Line" → tap the "To Union Station" result directly (skips a step vs v2).
  • Swipe the card left/right, watch map colors + markers swap.
  • Tap 🔍 above timeline, type "Peoria".
  • Tap FAB → Alerts → search "15L", click a route badge.
  • Tap ✕ on the card to return to Phase 1.
